== Description ==

With this plugin you can create upto 6 equal width column content on your WordPress blog and website.

You can also create different width columns as : one column with: 50% width and the remaining two with 25% width. You can create any number of such combinations which sum up to 100% width, all you need to do is use proper CSS class :)

Demo : http://icustomizethesis.com/content-column-classes/ to check the demo of this plugin. 
"How To" guide : http://icustomizethesis.com/how-to-use-genesis-column-classes-plugin/

== Installation ==

To install the plugin follow these steps :

1. Download the genesis-content-column-classes.zip file to your local machine.
2. Unzip the file 
3. Upload "genesis-content-column-classes" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress

That's it.

== Frequently Asked Questions ==

How many columns can I create ? 
You can create maximum of six column content. 

How are the widths of columns adjusted ? 
When you create 2 or more columns, the width of content column is equally distributed among the columns.

Can I create combination of columns like  3 columns with one column as 50% width and two other having 25% width each ? 
Yes, you can! You simply need to select the correct CSS class.
